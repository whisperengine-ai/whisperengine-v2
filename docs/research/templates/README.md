# Research Templates

Standard templates for research documentation to maintain consistency.

## Available Templates

- **[DAILY_LOG.md](DAILY_LOG.md)** - Quick daily observation entry (5-10 min)
- **[WEEKLY_SUMMARY.md](WEEKLY_SUMMARY.md)** - Weekly pattern synthesis (20-30 min)
- **[EXPERIMENT.md](EXPERIMENT.md)** - Formal hypothesis-driven experiment structure
- **[SESSION_LOG.md](SESSION_LOG.md)** - Structured logging for research sessions

## Usage

Copy the template you need and fill it out:

```bash
# Daily log
cp docs/research/templates/DAILY_LOG.md docs/research/journal/2025-12/2025-12-15.md

# Weekly summary
cp docs/research/templates/WEEKLY_SUMMARY.md docs/research/journal/2025-12/week-03.md

# New experiment
cp docs/research/templates/EXPERIMENT.md docs/research/experiments/E002_your_experiment.md
```

## Template Philosophy

Templates provide:
- **Structure** - Consistent organization across reports
- **Prompts** - Reminders of what to capture
- **Speed** - Reduce decision fatigue during documentation

But they're not rigid:
- Skip sections that don't apply
- Add sections if needed
- Let the template guide, not constrain
